# Get 命令 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为终端版 AI Sync Manager 增加 `ai-sync get <app> <path>` 与 `--edit`，支持目录浏览、文件查看和内置终端编辑。

**Architecture:** 在现有 `runtime + usecase + Cobra + Bubble Tea` 结构上扩展。普通 `get` 路径走 Cobra 输出，`--edit` 单独启动 Bubble Tea 编辑器；真实文件访问和安全校验集中放在 `internal/service/tool/accessor.go`，避免 CLI 直接读写文件系统。

**Tech Stack:** Go 1.25、Cobra、Bubble Tea、Bubbles `textarea/help/key`、Go 标准库 `os/filepath`
