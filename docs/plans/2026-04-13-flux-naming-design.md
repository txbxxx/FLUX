# Flux 命名设计

## 背景

原项目名 `AToSync` 和 `fl-manager` 较长且不够直观，需要一个简洁有力的新名字。

## 命名决策

| 项目 | 决策 |
|------|------|
| 产品名 | **Flux** |
| CLI 命令 | `fl` |
| 含义 | 流动/变化 |
| Tagline | AI 配置流动管理器 |

## 为什么是 Flux

1. **简洁有力** — 4 个字母，极简主义
2. **语义贴切** — "流动/变化"完美契合配置同步、模型切换的动态本质
3. **技术感强** — 已被广泛使用（React Flux 架构），程序员一看就懂
4. **无歧义** — 发音清晰，不会混淆
